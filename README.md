# Officially Cloned Tylerwise

A modern Next.js lead generation site built with 2025 design trends, featuring the Claude Money Making Prompts page.

## 🚀 Features

- **Modern Landing Page** - Hero section with email capture, feature cards, social proof, and CTAs
- **Claude Money Making Prompts Page** - Dedicated page with all 10 prompts:
  - FIND YOUR FIRST $1,000 SKILL
  - BUILD A FREELANCE OFFER IN 10 MINUTES
  - WRITE YOUR COLD OUTREACH SEQUENCE
  - CREATE A DIGITAL PRODUCT FROM SCRATCH
  - REVERSE-ENGINEER A PROFITABLE NICHE
  - BUILD A CONTENT STRATEGY THAT ATTRACTS CLIENTS
  - TURN YOUR KNOWLEDGE INTO A PAID NEWSLETTER
  - MAP OUT A 90-DAY INCOME PLAN
  - AUTOMATE YOUR FOLLOW-UP SYSTEM
  - CREATE YOUR PERSONAL BRAND IN ONE HOUR
- **Lead Gen Elements** - Email capture forms, CTAs, social proof sections
- **2025 Design Trends** - Glassmorphism, gradient text, floating animations, dark theme
- **Responsive Design** - Works on all devices

## 🛠️ Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React Icons
